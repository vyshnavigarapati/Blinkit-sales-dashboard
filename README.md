# Blinkit Sales Analysis Dashboard (Power BI)

## Project Overview
The Blinkit Sales Analysis Dashboard is an end-to-end data analytics project built using Power BI.  
It focuses on analyzing sales performance, customer ratings, outlet characteristics, and product trends to generate meaningful business insights and support data-driven decision making.

## Objectives
- Understand overall sales performance and average sales value
- Identify top-performing item categories and outlet types
- Analyze outlet size, location, and establishment trends
- Evaluate customer ratings and product demand patterns
- Provide clear visual insights through an interactive dashboard

## Dataset Information
The dataset contains sales records of Blinkit grocery outlets including:
- Item type and category  
- Outlet size, type, and location  
- Total sales and average sales  
- Customer rating and number of items  
- Establishment year and trend data  

## Tools & Technologies Used
- **Power BI** – Data modeling, DAX measures, dashboard creation  
- **Excel / CSV** – Raw dataset and preprocessing  
- **Data Cleaning & Transformation** – Handling missing values and formatting  

## Key Insights Generated
- Comparison of **total sales across outlet sizes and locations**
- Identification of **high-revenue item categories**
- Trend of **sales growth by outlet establishment year**
- Relationship between **customer ratings and sales performance**
- Distribution of **items across outlet tiers and formats**

## Dashboard Features
- Interactive filters for outlet location, size, and item type  
- KPI cards showing total sales, average sales, ratings, and number of items  
- Bar charts, donut charts, and trend lines for easy visualization  
- Clean and business-friendly dashboard layout  

## Business Impact
This dashboard helps stakeholders:
- Track performance of different outlet categories  
- Improve inventory and product planning  
- Identify low-performing segments  
- Make strategic business decisions using data insights
